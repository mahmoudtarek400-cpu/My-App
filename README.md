# Mahmoud Calculator (Flutter)

A simple Flutter calculator app with a small animated history graph.

## Requirements
- Flutter SDK (stable)
- Dart (comes with Flutter)
- For Android run:
  - Android Studio + Android SDK
  - An Android emulator or a real device with USB debugging
